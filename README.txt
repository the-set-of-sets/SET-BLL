In diesem Ordner finden Sie alle Programme, die ich für meine BLL "Eine mathematische Analyse des Kartenspiels SET" geschrieben und verwendet habe.

20set.gap: Findet eine SET-freie 20-elementige Menge.
AGL.gap: Die Gruppe AGL wird in GAP hergestellt.
alle.gap: Die Anzahl von Bahnen von AGL(4,3) (Symmetriegruppe) auf P_k(F_3^4) (Menge aller k-elementigen Teilmengen von Karten) wird bestimmt.
alle-daten.gap: Daten von alle.gap.
bilder.asy: Anymptote-Programm, mit dem ich die Abbildungen von SET-Karten erstellt habe.
eins-reps.gap: Daten von null-eins.gap, nämlich alle Repräsenanten von Mengen mit genau einem SET.
null-eins.gap: Berechnung aller Repräsentanten von Mengen mit 1) keinen SETs 2) genau einem SET.
null-reps.gap: Daten von null-eins.gap, nämlich alle Repräsentanten von Mengen mit keinen SETs.
